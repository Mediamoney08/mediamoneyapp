# User Guide: Profile Settings & Security

## Quick Access

**Navigate to Profile Settings:**
1. Click your profile icon in the top-right corner
2. Select "Profile Settings" from the dropdown menu
3. You'll see 4 tabs: Profile, Security, 2FA, and Activity

## üîí Security Tab

### Change Your Password

**Steps:**
1. Go to **Security** tab
2. Enter your **current password**
3. Enter your **new password** (minimum 8 characters)
4. **Confirm** your new password
5. Click **Update Password**

**Tips:**
- Use a strong, unique password
- Mix uppercase, lowercase, numbers, and symbols
- Don't reuse passwords from other sites
- Consider using a password manager

### Change Your Email

**Steps:**
1. Go to **Security** tab
2. Scroll to "Change Email" section
3. Enter your **new email address**
4. Click **Request Email Change**
5. Check your **new email** for verification link
6. Click the verification link to complete the change

**Important:**
- You'll receive emails at both old and new addresses
- Your old email remains active until you verify the new one
- The verification link expires after 24 hours

## üõ°Ô∏è 2FA Tab (Two-Factor Authentication)

### What is 2FA?

Two-Factor Authentication adds an extra security layer by requiring:
1. Your password (something you know)
2. A code from your phone (something you have)

### Enable 2FA

**Step 1: Start Setup**
1. Go to **2FA** tab
2. Click **Enable 2FA** button

**Step 2: Download Authenticator App**

Choose one of these apps (if you don't have one):
- **Google Authenticator** (iOS/Android)
- **Microsoft Authenticator** (iOS/Android)
- **Authy** (iOS/Android/Desktop)
- **1Password** (iOS/Android/Desktop)

**Step 3: Scan QR Code**
1. Open your authenticator app
2. Tap "Add Account" or "+"
3. Scan the QR code shown on screen
4. The app will start generating 6-digit codes

**Step 4: Verify**
1. Enter the 6-digit code from your app
2. Click **Verify and Enable**
3. 2FA is now active! üéâ

**Step 5: Save Backup Codes**
1. You'll see 10 backup codes
2. Click **Download** to save them as a file
3. Or click **Copy** to copy them
4. Store them in a safe place (not on your phone!)

**Why Backup Codes?**
- Use them if you lose your phone
- Each code works only once
- Keep them safe and secure

### Disable 2FA

**Warning:** This makes your account less secure!

1. Go to **2FA** tab
2. Click **Disable 2FA** button
3. Confirm the action
4. 2FA is removed

## üîê Security Best Practices

### 1. Enable 2FA
**Why:** Protects your account even if password is stolen  
**How:** Follow the 2FA setup guide above  
**When:** Right now! Don't wait

### 2. Use Strong Passwords
**Why:** Harder for hackers to guess  
**How:** 
- At least 8 characters
- Mix of uppercase and lowercase
- Include numbers and symbols
- Don't use personal information

**Examples:**
- ‚ùå Bad: `password123`, `john1990`
- ‚úÖ Good: `Tr0pic@l$unset!42`

### 3. Save Backup Codes
**Why:** Access your account if you lose your phone  
**How:** Download and store securely  
**Where:** 
- Password manager
- Encrypted file
- Physical safe
- NOT on your phone!

### 4. Monitor Your Activity
**Why:** Detect unauthorized access early  
**How:** Check Activity tab regularly  
**When:** At least once a week

### 5. Keep Email Updated
**Why:** Receive important security notifications  
**How:** Update in Security tab  
**When:** Whenever you change email addresses

### 6. Use Trusted Devices
**Why:** Reduce risk of keyloggers and malware  
**How:** 
- Only login on your own devices
- Avoid public computers
- Use secure networks (not public WiFi)

## üÜò Common Issues & Solutions

### Issue: Lost My Phone (with Authenticator)

**Solution:**
1. Use a backup code to login
2. Go to Profile Settings ‚Üí 2FA tab
3. Disable 2FA
4. Set up 2FA again on your new device

**Prevention:**
- Always save backup codes
- Consider using Authy (cloud backup)
- Set up 2FA on multiple devices

---

### Issue: Forgot My Password

**Solution:**
1. Click "Forgot Password" on login page
2. Enter your email address
3. Check email for reset link
4. Click link and set new password
5. Login with new password

**If you have 2FA enabled:**
- You'll still need your authenticator code
- Or use a backup code

---

### Issue: Can't Access Email

**Solution:**
1. If you can still login:
   - Go to Security tab
   - Change email to one you can access
   - Verify new email

2. If you can't login:
   - Contact support with proof of identity
   - Support will help recover your account

**Prevention:**
- Keep email address current
- Use a reliable email provider
- Have a backup email

---

### Issue: Backup Codes Not Working

**Possible Causes:**
- Code already used (each works once)
- Typing error
- 2FA was reset

**Solution:**
1. Try another backup code
2. Check for typos
3. If all codes fail, contact support

---

### Issue: Email Verification Not Received

**Solution:**
1. Check spam/junk folder
2. Wait a few minutes (can take up to 10 minutes)
3. Request new verification email
4. Try different email address
5. Contact support if still not received
