## How to Access Admin Dashboard

### Step 1: Login

2. Enter the admin email: `mediamoney@miaoda.com`

Or click on your profile icon in the header and look for admin options.

## Admin Dashboard Features

The admin dashboard provides access to:
- **User Management:** View and manage all users
- **Payment Requests:** Review and approve/reject payment requests
- **Order Management:** View and manage all orders
- **Product Management:** Add, edit, and delete products
- **Category Management:** Manage service categories
- **Payment Methods:** Configure payment methods
- **Stock Management:** Manage gift card inventory
- **Analytics:** View site statistics and reports
- **Settings:** Configure site settings

## Security Notes

- The admin account has full access to all features
- Admin routes are protected and require admin role
- Regular users cannot access admin pages
- All admin actions are logged for security

## Troubleshooting

**"Access Denied" Error:**
- Make sure you're logged in with the admin account
- Check that your profile role is set to 'admin'
- Try logging out and logging back in

**Can't See Admin Options:**
- Verify you're using the correct admin email
- Check browser console for any errors
- Clear browser cache and try again
